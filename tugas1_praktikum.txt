1.jelaskan apa yang dimaksud dengan dokumen root
= konteks sistem manajemen dokumen, pengembangan web, atau struktur data. Secara umum, dokumen root merujuk pada dokumen atau file utama yang menjadi titik awal atau dasar dari suatu struktur dokumen atau sistem.
2.pada apache web server,apakah dokumen root harus terletak dalam direktori C;/xampp/htdocs? jelaskan
= Pada Apache Web Server yang diinstal melalui XAMPP, dokumen root secara default terletak di direktori C:\xampp\htdocs. Ini adalah lokasi di mana Apache mencari file-file yang akan disajikan kepada pengguna ketika mereka mengakses server web. Namun, dokumen root tidak harus selalu berada di direktori tersebut
3.jelaskan apa yang dimaksud dengan perintah git:init,clone,remote,add,commit, push dan chechkout
= git init
Perintah ini digunakan untuk menginisialisasi repositori Git baru
git clone
Perintah ini digunakan untuk membuat salinan lokal dari repositori yang sudah ada
git remote
Perintah ini digunakan untuk mengelola remote repository yang terhubung dengan repositori lokal
git add
Perintah ini digunakan untuk menambahkan perubahan (file baru, file yang dimodifikasi, atau file yang dihapus) ke staging area
git commit
Perintah ini digunakan untuk menyimpan perubahan yang telah ditambahkan ke staging area ke dalam repositori
git push
Perintah ini digunakan untuk mengirimkan commit yang telah dibuat di repositori lokal ke repositori remote
git checkout
Perintah ini digunakan untuk berpindah antara cabang (branch) dalam repositori git

5.apa bedanya variable user dengan variable system
= Variabel Pengguna (User Variables)
Variabel pengguna adalah variabel yang didefinisikan oleh pengguna atau aplikasi tertentu
 sedangkan Variabel Sistem (System Variables)
 Variabel sistem adalah variabel yang didefinisikan oleh sistem operasi dan berlaku untuk seluruh sistem
6.apa persamaan variable system dengan variable constant
= variabel sistem dan variabel konstan digunakan dalam konteks yang berbeda (sistem operasi vs. pemrograman), keduanya memiliki kesamaan dalam hal nilai yang tidak berubah, penggunaan untuk konfigurasi, akses global, dan dukungan untuk stabilitas. Variabel sistem menyimpan informasi penting untuk operasi sistem, sementara variabel konstan digunakan dalam pemrograman untuk menjaga nilai tetap yang diperlukan dalam logika aplikasi
